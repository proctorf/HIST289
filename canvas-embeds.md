## Instructions for Canvas Implementation

1. **Copy the iframe code** for the week you want to embed
2. **In Canvas**, navigate to your course and go to Modules
3. **Add a new item** to the appropriate module
4. **Select "External URL"** as the item type
5. **Paste the iframe code** into the URL field, or use the HTML editor
6. **Adjust height values** as needed based on content length
7. **Save and publish** the module item

## Embed Code Customization

You can adjust the iframe parameters:
- `width="100%"` - Makes the iframe responsive
- `height="600"` - Adjust height as needed (try 400-800px)
- `frameborder="0"` - Removes iframe border
- Add `scrolling="yes"` if you want to ensure scrolling is enabled
